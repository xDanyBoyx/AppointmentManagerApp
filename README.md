# Agenda Virtual de Clientes y Citas

Aplicación móvil desarrollada en Flutter para administrar clientes y sus citas.

## Características

- Registro de clientes
- Registro de citas
- Actualización de información
- Eliminación de registros
- Persistencia local con SQLite
- Relaciones mediante llaves foráneas

## Tecnologías

- Flutter
- Dart
- SQLite (sqflite)
